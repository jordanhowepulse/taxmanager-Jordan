Rolling 12-Month Budget

Budget periods follow the 28th-to-27th cycle — June 2026 covers 28 May to 27 June 2026. The window auto-rolls forward when a new month starts so the current period is always at the left. Click any budget cell to override that single month (cells highlight amber). Use the Edit dialog on each category for forward-from-month bulk changes.

Categories — Central Manager

Single place to view, edit and remove every budget line. Edit names or amounts inline; changes flow to the Budget table, Dashboard, Forecast and Import suggestions. Use the dedicated tabs for richer fields (due date, account goal, etc).
